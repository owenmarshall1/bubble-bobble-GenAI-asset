## Overview

This assignment does not use any generative AI. All sprites, visual assets, sound effects, and music was created manually using traditional tools

- [x] Player sprite replaced
- [x] Enemy sprite replaced
- [x] Tileset used was replaced
- [x] Background was replaced
- [x] All (6) sound effects were recorded by me
- [x] Background music I made with the help of a friend who is very good at making music because I am not
- [x] UI was replaced

At first I was a little uncomfortable using generative AI to create artwork for the game, and through a few trials I became aware very quickly that it really sucked and I felt wasn't going to satisfy me when making the project. 
I hope you are able to tell the passion and effort I put into manually creating the art for the project, becuase I feel using generative AI for artwork strips any project of its artistic value. Using AI in game development isn't bad when used for things like code or frame generation
but especially in the state it's in now I do not believe it is worth being used to create assets.
